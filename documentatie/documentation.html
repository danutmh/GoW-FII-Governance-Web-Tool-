Documentation for the GoW project implemented by Munteanu Cosmin and Mihai Dumitru

Technology stack

Our project is composed of front-end and back-end.

The front-end is realised using mainly html and css, with some javascript that implements some functionalities to our webpage (such as the login module).

The back-end is implemented using php. Our database (that will store the accounts of our users) is going to use the mysqli extension. The administrator is allowed to make CRUD operations on the database, (alter accounts, add/remove documents, add messages in the Atom feed) through a graphic interface. Teachers will be allowed to add comments on each document or message. Students will be allowed to read those documents and messages. Unregistered users won't have acces to the Atom feed and faculty-related documents. The Atom feed (uses XML) contains content available only to members of the faculty.
